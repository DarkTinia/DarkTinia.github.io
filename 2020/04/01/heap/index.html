静态分析

程序一开始输出一个菜单，然后根据输入调用不同的函数。重点在create和edit

c
c
c

create会先malloc0x10大小的空间用于存放用户申请的heap的信息，该空间的地址存储在数组heaparray中。该空间前0x10存size，后0x10存地址，而edit则直接根据该空间存储的地址来修改内容。也就是说如果能任意修改这个地址，就可以实现任意地址读写。
程序调用my_read存在off-by-one，可以根据这一点修改下一个heap的size。
修改后free掉，再create一个适当大小的heap就可以修改heaparray中的heap里存储的地址
